## Setup .env File

Create a `.env` file in the root directory of your project and include the following environment variables:

```env
PORT=...
MONGODB_URL=...
JWT_SECRET=...
NODE_ENV=...
VITE_API_URL=...
```

### Build the app

``` shell
yarn build
```

### Start the app

``` shell
yarn start
```
